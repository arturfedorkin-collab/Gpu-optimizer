Dota 2 GPU Optimizer

Подбор оптимальных настроек Dota 2 для хорошего FPS и приятной картинки

Параметры запуска DOTA 2

-novid -high -map dota -prewarm -nod3d9ex -nohltv -novr -dx11 +dota_embers 0 +map_enable_background_maps 0
-novidПропускает заставку Valve при запуске.
-highСтавит Dota 2 высокий приоритет.
-map dotaПредзагружает карту Dota 2.
-prewarmЗаранее загружает ресурсы и уменьшает фризы.
-nod3d9exМожет помочь старым системам.
-nohltvОтключает лишние функции трансляции.
-novrОтключает поддержку VR.
-dx11Лучший вариант для NVIDIA.
-vulkanМожно использовать для AMD или Linux.
+dota_embers 0Отключает частицы в главном меню.
+map_enable_background_maps 0Отключает анимированный фон меню.
