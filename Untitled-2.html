Параметры запуска DOTA 2

-novid -high -map dota -prewarm -nod3d9ex -nohltv -novr -dx11 +dota_embers 0 +map_enable_background_maps 0
-novid Пропуск заставки Valve при запуске игры, ускоряет загрузку.
-high Устанавливает высокий приоритет Dota 2, улучшает плавность.
-map dota Предзагружает карту, сокращает загрузку матча.
-prewarm Загружает ресурсы заранее, уменьшает фризы.
-nod3d9ex Улучшает производительность в оконном режиме.
-nohltv Отключает HLTV, освобождает ресурсы.
-novr Отключает поддержку VR, уменьшает использование RAM.
-dx11 / -vulkan DX11 — лучше для NVIDIA, Vulkan — лучше для AMD/Linux.
+dota_embers 0 Убирает частицы в главном меню.
+map_enable_background_maps 0 Отключает анимированное меню, снижает нагрузку на GPU.
